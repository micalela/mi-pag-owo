cómo montar discos automáticamente al bootear:

  • sudo blkid --> ver información sobre los discos
  • crear carpeta para que se monte el disco: sudo mkdir /mnt/nombre-del-disco (lo que está después de /dev/ en el directorio)
  • abrir nano para editar el archivo /etc/fstab: sudo nano /etc/fstab
  • agregar una línea al final con la info del disco que quiero que se monte:

    UUID=uuid-del-disco *tab* carpeta que hice antes *tab* formato del disco (ntfs, ext4) *tab* defaults *tab* 0 *tab* 2

    el default es para mount option, el 0 es para dump, el 2 para pass
  • enjoy new auto mounting drives
